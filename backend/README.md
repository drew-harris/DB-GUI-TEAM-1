## Current routes
  * API Docs available at ```http://localhost:8000/docs```
  * Add ```PRIVATE_KEY``` to .env in backend dir
  * Import ```postman_collection.json``` and ```postman_environment.json``` in ```postman``` dir to use the current routes in Postman.
